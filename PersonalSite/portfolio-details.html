Unipop - Ecommerce site

  • Category: Full-Stack Website
  • Tech Stack: : React, Node, Redux, Commerce, Stripe
  • Project date: Summer 2022
  • Project URL: Git repo

The concept and functionality of UniPop

With this website you are able to manage your own online store by using an easy database system called Commerce that works well with monitoring products and customer insights, after posting products users can buy them and receive updates on when they are coming. The website has back-end for managing the cart functionality for each user managing it, the payment system uses Stripe to accept Visa and Paypal etc. In the future it would be a giant amazon style website where you can post products and buy from others.
